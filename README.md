# Julia

Julia on Ruby.

## Supported versions

### Ruby

- 3.1

### Julia

- 1.8
- 1.7
- 1.6

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'julia'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install julia

## Usage

```ruby
require 'julia'

Julia.eval('print(pi)')
```
